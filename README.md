# NuxtHub Starter

A template to skip boring setup

## Features

- NuxtHub
- Drizzle
- I18n
- @nuxt/ui
- nuxt-auth-utils
- Clean Architecture
- Login (OAuth Providers, MagicLink)
- Settings (avatar and account deletion)

## Setup

- create Google credentials
- install dependancies

```bash
pnpm install
```

- start dev server

```bash
pnpm run dev
```
